# 树莓派小车
1. 打开树莓派小车，等待，直到听到”滴“一声
2. 观察小车电路板上的灯，若为慢速闪烁，则为直连模式，需用app调至局域网模式
3. 在局域网模式下（电路板上的灯快速闪烁），小车跟电脑连接同一个wifi后，此时小车电路板上的灯常亮（停止闪烁）
4. 通过vscode的ssh-remote插件连接小车（小车的ip地址通过开热点的手机查看，ssh密码为123456）
5. 进入小车的HiwonderSDK文件夹
6. 使用sudo python3 receiver.py指令，运行receiver.py文件
